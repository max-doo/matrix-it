# MatrixIt UI 设计系统（MVP）

## 视觉基调

- 风格：轻快、简约、高级（卡片 + 充足留白 + 克制阴影）
- 信息密度：列表默认“舒适”，支持后续切换“表格密集编辑”
- 动效：150–300ms 微交互，尊重 prefers-reduced-motion

## 颜色（按 PRD）

- 主色（强调/主按钮/进度）：Tiffany Blue `#0ABAB5`
- 背景：`#F5F7FA`
- 卡片：`#FFFFFF`
- 文本：主文本 `#0F172A`，次文本 `#475569`
- 边框：`#E2E8F0`
- 状态：
  - 成功：`#10B981`
  - 警告/失败：`#F43F5E`
  - 中性：`#CBD5E1`

## 字体

- 默认：系统无衬线（Windows：Segoe UI / 等效字体）
- 字号建议：
  - 页面标题：16–18px（Semibold）
  - 列表主信息：14px（Medium）
  - 辅助信息/说明：12px（Regular）

## 可用性（来自 UI/UX 规则库）

- 可点击元素最小触达区域：44×44
- 仅图标按钮必须有 aria-label；键盘可导航且 focus ring 可见
- 异步操作按钮：加载中禁用 + 明确反馈（进度/状态）
- 颜色不是唯一信息载体：状态使用点/文本同时表达
