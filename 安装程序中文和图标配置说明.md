## 当前配置文件内容

### tauri.conf.json
```json
{
  "bundle": {
    "icon": [
      "icons/icon.ico",
      "icons/icon.png"
    ],
    "windows": {
      "wix": {
        "language": {
          "zh-CN": {
            "localePath": "wix/locales/zh-CN.wxl"
          }
        }
      },
      "nsis": {
        "displayLanguageSelector": false,
        "languages": ["SimpChinese"]
      }
    }
  }
}
```

## 优化建议

### 如果 MSI 中文未生效
可以通过以下方式强制设置默认语言（需要自定义 WiX 模板）：

1. 创建自定义 WiX 主模板（高级）
2. 添加 `Product.Language` 属性设置为 `2052`（简体中文）
3. 配置 `tauri.conf.json` 使用自定义模板

**示例**（需要自定义模板）：
```xml
<Product Language="2052" ...>
```

## 图标显示故障排除

### 问题：安装包图标未显示
**可能原因**：
1. Windows 资源管理器图标缓存未更新
2. 图标文件分辨率不足

**解决方案**：
1. 刷新资源管理器（F5）或重启资源管理器
2. 清除图标缓存：
   ```powershell
   # 清除图标缓存（管理员权限）
   taskkill /f /im explorer.exe
   del /a /q "%LocalAppData%\IconCache.db"
   start explorer.exe
   ```
3. 确保 `icon.ico` 包含多种尺寸（16x16, 32x32, 48x48, 256x256）
