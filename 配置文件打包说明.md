# 配置文件打包说明

## 概述

MatrixIt 使用 **脱敏默认配置** (`config.default.json`) 进行打包，确保：
1. 不泄露开发环境的敏感信息（API 密钥、飞书配置等）
2. 用户首次打开应用时有完整的字段定义和合理的默认设置
3. 用户误删配置后可自动恢复

## 配置文件结构

### 打包资源
```
src-tauri/resources/
├── jcr.db               # 期刊数据库
└── config.default.json  # 脱敏默认配置（包含完整 fields 定义）
```

### 用户配置
```
安装目录/
├── config/
│   └── config.json      # 用户配置（首次启动自动生成）
└── resources/
    └── config.default.json  # 默认配置（只读模板）
```

## 配置加载优先级

1. **用户配置** `config/config.json`（优先读取，可修改）
2. **默认配置** `resources/config.default.json`（用户配置不存在时自动复制恢复）

## 自动恢复机制

当用户删除 `config/config.json` 后重启应用：
- Rust 端 `resolve_config_path()` 检测到用户配置不存在
- 自动从 `resources/config.default.json` 复制到 `config/config.json`
- 用户可重新配置 API 密钥等信息

## 默认配置内容

`config.default.json` 包含：

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| `fields` | 完整定义 | 所有字段的中文名映射 |
| `llm.provider` | `custom` | LLM 提供商设为自定义 |
| `llm.api_key` | 空 | 需用户自行配置 |
| `feishu.*` | 空 | 飞书配置需用户自行配置 |
| `zotero.data_dir` | 空 | 前端自动识别 `%USERPROFILE%\Zotero` |
| `ui.debug_llm` | `false` | 生产环境默认关闭调试 |

## 开发与打包流程

### 修改默认配置

1. 编辑 `src-tauri/resources/config.default.json`
2. **注意**：绝对不要在此文件中添加任何敏感信息

### 打包

```powershell
# 1. 构建 Sidecar
.\scripts\build_sidecar.ps1

# 2. 打包应用
cd src-tauri
cargo tauri build
```

### 验证

安装后检查：
- [ ] `resources/config.default.json` 存在且无敏感信息
- [ ] 首次启动自动生成 `config/config.json`
- [ ] 设置页字段显示中文
- [ ] 删除 `config/config.json` 后重启可恢复

## 与开发环境的区别

| 环境 | 配置路径 | 说明 |
|------|---------|------|
| 开发 | `config/config.json` | 可包含开发密钥（已加入 .gitignore） |
| 打包 | `resources/config.default.json` | 脱敏模板，自动复制给用户 |

## 安全注意事项

> ⚠️ **严禁提交敏感信息**
> - `config/config.json` 中的 API 密钥不应提交到 Git
> - 使用 `config/config.local.json` 存放个人密钥（已在 .gitignore）
> - `src-tauri/resources/config.default.json` 必须保持脱敏状态
